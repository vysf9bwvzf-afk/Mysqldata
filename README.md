Dit document geeft een overzicht van het Entity Relationship Diagram (ERD) voor Breeze Net. Het model beschrijft alle entiteiten die nodig zijn binnen het reserverings- en beheersysteem, inclusief hun belangrijkste attributen en onderlinge relaties. 
Het ERD vormt de basis voor de SQL-tabellen, API-koppelingen en dashboardvisualisaties.
Het doel van dit ERD is om een duidelijke datastructuur te bieden waarmee alle onderdelen van het project (Business, Software, Media en Infra) kunnen samenwerken op basis van één consistente dataset.
